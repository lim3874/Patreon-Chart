# v2.0.0 - Creator Analytics Redesign

Released: 2026-05-12

## 변경 사항

- 데스크톱 앱 UI를 Creator Analytics 스타일로 전면 재구성했습니다.
- 기존 상단 탭 구조를 제거하고 왼쪽 사이드바 내비게이션으로 변경했습니다.
- 상단에는 `Analytics Engine`, `Date Range`, `Import from Gmail`, 설정/폴더/Patreon 설정 바로가기를 배치했습니다.
- `Summary` 화면을 카드형 지표와 티어 분포 차트 중심으로 다시 구성했습니다.
- `Period` 화면을 큰 차트 카드와 Daily/Weekly/Monthly 세그먼트 컨트롤 중심으로 재구성했습니다.
- `List` 화면을 검색, 티어 필터, 상태 필터가 있는 현대적인 테이블 화면으로 변경했습니다.
- `Patreon API` 화면도 같은 카드형/테이블형 디자인에 맞춰 재구성했습니다.
- 제공된 디자인과 일관되도록 앱을 다크 전용 인터페이스로 고정했습니다.

## 추가 사항

- `List` 화면에 이름/이메일/제목 검색 기능을 추가했습니다.
- `List` 화면에 티어 필터와 결제 상태 필터를 추가했습니다.
- 상단 액션바에서 날짜 범위 설정, Gmail 가져오기, 설정 파일, 결과 폴더, Patreon API 키 설정을 바로 열 수 있습니다.
