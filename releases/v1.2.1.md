# v1.2.1 - Chart App Icon

Released: 2026-05-12

## 추가 사항

- 차트 느낌의 전용 앱 아이콘을 PNG/ICO 형식으로 추가했습니다.
- 프로그램 창과 Windows 작업표시줄에 전용 아이콘이 표시되도록 적용했습니다.
- 바탕화면 바로가기에서도 같은 아이콘을 사용할 수 있도록 아이콘 자산을 포함했습니다.

## 변경 사항

- Windows AppUserModelID를 지정해 실행 중인 앱이 Python/Tk 기본 아이콘 대신 Patreon Chart 전용 아이콘으로 보이도록 조정했습니다.

## 참고

- 이미 작업표시줄에 고정해둔 바로가기는 Windows 아이콘 캐시 때문에 즉시 바뀌지 않을 수 있습니다. 그 경우 작업표시줄 고정을 해제한 뒤 바탕화면 바로가기를 다시 고정하면 새 아이콘이 확실히 적용됩니다.
