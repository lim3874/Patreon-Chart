# v2.1.1 - High-DPI clarity patch

## Fixed

- Enabled Windows per-monitor DPI awareness before the Tk window is created, which prevents Windows from bitmap-scaling the whole app on 4K/high-DPI monitors.
- Added Tk DPI-based font scaling so dashboard text is rendered by Tk at the active monitor resolution.
- Replaced the Windows taskbar icon with a sharper high-contrast chart icon and packaged explicit 16, 24, 32, 48, 64, 128, and 256 px frames.
- Switched the app to the new v3 icon files.
