# v2.3.0 - Summary period and tier sort controls

- Added `멤버순` and `티어순` sort controls to the tier distribution chart.
- Saved the selected tier distribution sort mode in local app settings.

- The `전체 회원` summary card now shows the actual selected date span, such as `2026/04/13 - 2026/05/12`, instead of `선택 기간`.
- Moved the `티어 1`, `티어 2`, and `티어 3` summary cards to the second row.
- Removed the `확인 필요` card from the summary card grid.
