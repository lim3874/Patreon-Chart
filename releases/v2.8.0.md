# v2.8.0 - Custom Dark Title Bar and Patreon Discord Fields

## Added

- Replaced the native Windows title bar with a custom dark title bar inside the app. This removes the remaining white title bar on systems where Windows/DWM ignores app dark-title requests.
- Added Patreon API columns for Discord ID and Discord name from `user.social_connections`.
- Added more Patreon member fields to the Patreon API export/table: next charge date, next pay amount, pledge cadence, gift/free-trial flags, member note, profile URL, and lifetime support amount.
- Added a unit test for Discord social connection parsing.

## Changed

- Patreon API table sorting now treats the new amount, date, and boolean fields as typed values instead of plain text.

## Verification

- `python -m py_compile patreon_member_gui.py patreon_api.py export_patreon_members.py`
- `python -m unittest discover -s tests`
- Local screenshot smoke test confirmed the app uses custom dark chrome and no native white title bar.
