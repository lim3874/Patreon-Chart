# v2.2.0 - Korean localization

## Added

- Localized the desktop dashboard UI into Korean.
- Added Korean labels for the sidebar navigation, top action bar, summary metrics, period chart controls, table filters, date range dialog, and Patreon API workspace.
- Added Korean display values for Gmail member records, including 신규 가입, 재구독, 티어, 청구 주기, 결제 상태, and 확인 필요.
- Added Korean display values for Patreon API member statuses, tier names, and common payment statuses.

## Changed

- Existing saved English settings are converted to Korean labels when the app starts, so older local settings keep working.
