# v1.2.2 - Sharper Taskbar Icon

Released: 2026-05-12

## 수정 사항

- 작업표시줄에서 아이콘이 저화질로 보이던 문제를 개선했습니다.
- 16, 24, 32, 48, 64, 128, 256 px 크기별로 따로 그린 프레임을 ICO 안에 넣어 작은 크기에서도 차트 모양이 선명하게 보이도록 했습니다.
- Windows에서는 큰 PNG를 축소하지 않고 `.ico` 파일을 직접 사용하도록 변경했습니다.
- 바탕화면 바로가기는 새 파일명 `patreon_chart_icon_v2.ico`를 보도록 업데이트했습니다.

## 참고

- Windows가 기존 아이콘을 캐시에 들고 있으면 한동안 예전 아이콘처럼 보일 수 있습니다. 앱을 완전히 종료한 뒤 다시 실행하거나, 작업표시줄 고정을 해제하고 바탕화면 바로가기를 다시 고정하면 새 아이콘이 적용됩니다.
