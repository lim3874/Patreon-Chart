# Patreon Chart v1.1.0

Feature release.

## Added

- Patreon-style date range preset dropdown:
  - 지난 24시간
  - 지난 30일
  - 지난 6개월
  - 지난 12개월
  - 전체
  - 사용자 지정
- Patreon-style period chart controls:
  - 전체
  - 멤버십 등급
  - 청구 주기
  - 결제 상태
  - 유료로 전환하는 경로
- Daily, weekly, and monthly chart granularity selection.
- Card-style summaries below the period chart for the selected insight category.
- Member table columns for:
  - 항목
  - 멤버십 등급
  - 청구 주기
  - 결제 상태
  - 유료 전환 경로

## Changed

- The period chart now uses a Patreon-like stacked bar view with category cards.
- Date range presets drive both display filtering and Gmail fetch query dates.
