# v1.2.0 - Patreon-Style Desktop Dashboard

Released: 2026-05-12

## 변경 사항

- 데스크톱 대시보드 디자인을 Patreon 화면에 가까운 다크 스타일로 재정비했습니다.
- 상단 컨트롤, 탭, 버튼, 카드, 테이블을 어두운 배경과 베이지 계열 테두리 중심으로 맞췄습니다.
- 신규 로컬 설정에서는 다크 모드가 기본으로 켜지도록 변경했습니다.
- 한국어 UI가 더 안정적으로 보이도록 주요 글꼴을 `Malgun Gothic` 기준으로 통일했습니다.
- 차트 카테고리 버튼의 선택/미선택 상태를 다크 모드에서도 구분되게 조정했습니다.

## 수정 사항

- 다크 모드에서 콤보박스, 날짜 입력칸, 드롭다운 목록, 표 헤더, 차트 라벨 일부가 흐리거나 안 보이던 문제를 개선했습니다.
- Patreon API 탭의 회원 목록 테이블이 전체 다크 테마와 더 일관되게 보이도록 조정했습니다.

## 참고

- 기존 `app_settings.json`에 다크 모드가 꺼진 상태로 저장되어 있으면 그 설정을 계속 존중합니다. 새 디자인을 바로 보려면 앱 오른쪽 위 `다크 모드`를 체크하면 됩니다.
