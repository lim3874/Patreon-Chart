# v2.5.0 - Patreon API Table Sorting

## Added

- Added clickable sorting to every Patreon API table header.
- Repeated clicks on the same Patreon API column now toggle ascending and descending order.
- Added sort markers to the active Patreon API column header.

## Changed

- Patreon API amount and date columns now sort by numeric/date values instead of plain text.
- Sorting changes only the visible table order and does not rewrite the saved CSV/API data.
