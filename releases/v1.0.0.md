# Patreon Chart v1.0.0

Initial public release for the local Patreon Gmail and Patreon API dashboard.

## Added

- Gmail API reader for Patreon new-member emails.
- New-member parser that extracts date, name, email, currency, amount, and message metadata.
- Tier classifier for 2.99, 4.99, 9.99, and 29.99 USD membership levels.
- Hybrid tier matching:
  - exact local currency price map first
  - historical/latest exchange-rate fallback
  - review flag for uncertain matches
- CSV, XLSX, and HTML report outputs.
- Desktop GUI with:
  - dark mode toggle
  - summary metrics
  - tier distribution chart
  - daily/weekly/monthly period chart
  - member table with tier/review filters
- Patreon API tab for current members:
  - status
  - current tier
  - entitled amount
  - last charge date/status
  - pledge relationship start
- Local settings dialog for Patreon API credentials.
- Windows desktop launcher support.
- README with Gmail API, Patreon API, and usage instructions.
- Parser unit tests.

## Changed

- The desktop shortcut now opens the GUI dashboard instead of only running the console exporter.

## Security

- `credentials.json`, `token.json`, `patreon_credentials.json`, generated output, and cache files are ignored.
- Example config and credential templates are included without real tokens.
