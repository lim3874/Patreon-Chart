# v2.12.0 - Patreon Table Column Controls

## Added

- Added drag-and-drop reordering for Patreon API table headers.
- Added `표 컬럼 설정` in the Patreon API workspace.
- Added a Patreon table column settings shortcut in the main settings dialog.
- Added persistent local settings for:
  - Patreon API table column order
  - Patreon API table visible columns
- Added a column settings dialog with:
  - show/hide toggle
  - move up/down controls
  - show all
  - reset to defaults

## Changed

- The Patreon API table now keeps all data columns internally and only changes the visible order through Treeview display columns. This keeps CSV parsing and table row mapping stable while allowing user-customized display.
