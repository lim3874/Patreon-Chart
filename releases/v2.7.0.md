# v2.7.0 - Calendar Date Range Picker

## Added

- Added calendar buttons next to the start and end date fields in the date range dialog.
- Selecting a date from the calendar automatically switches the range mode to `사용자 지정`.

## Changed

- Simplified the date range presets to `지난 30일`, `반년`, `1년`, `전체`, and `사용자 지정`.
- Older saved labels such as `지난 6개월` and `지난 12개월` are mapped to the new preset names.
