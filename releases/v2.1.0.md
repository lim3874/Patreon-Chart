# v2.1.0 - Rejoin Analytics

Released: 2026-05-12

## 추가 사항

- 같은 이메일 또는 이메일이 없는 경우 같은 이름이 Patreon 신규 가입 메일에 여러 번 등장하면 재구독 의심 이벤트로 탐지합니다.
- `Summary` 화면에 `REJOINED` 카드가 추가되었습니다.
- `Period` 화면에 `Rejoins` 차트 분류가 추가되어 첫 가입과 재구독 의심 이벤트를 기간별로 비교할 수 있습니다.
- `List` 화면에서 재구독 의심 행은 `CATEGORY`가 `Rejoin`으로 표시됩니다.
- `List` 화면의 상태 필터에 `Rejoined`를 추가해 해당 회원만 빠르게 확인할 수 있습니다.

## 판정 방식

- 전체 Gmail 가입 기록을 기준으로 같은 회원의 첫 가입 이후 다시 들어온 가입 메일을 `Rejoin`으로 봅니다.
- 차트와 카드의 숫자는 현재 선택한 날짜 범위 안에 들어오는 `Rejoin` 이벤트만 집계합니다.
- Patreon API의 현재 스냅샷만으로는 과거 취소 이력을 확정하기 어렵기 때문에, 이 기능은 Gmail 가입 메일 누적 기록을 기준으로 동작합니다.
