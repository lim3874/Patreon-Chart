# v2.11.0 - In-App Settings Dialog

## Added

- Added a proper in-app settings dialog for the top-bar settings icon.
- Added settings sections for:
  - Gmail search query
  - tier USD prices
  - local currency price maps
  - fallback exchange rates
  - excluded emails
  - Patreon API settings shortcut
  - Discord bot token and server ID
- Added a `config.json 고급 편집` shortcut inside the new settings dialog for manual troubleshooting.

## Changed

- The top-bar settings icon no longer opens `config.json` directly in Notepad.
- Discord bot credentials can now be saved from the main settings dialog or the dedicated Discord bot dialog.
